c++基础入门注意事项
2022-03-10 23:36:00 # C++

  1. switch(条件):

    • 注意1：switch语句中表达式类型只能是整型或者字符型
    • 注意2：case里如果没有break，那么程序会一直向下执行
    • 总结：与if语句比，对于多条件判断时，switch的结构清晰，执行效率高，缺点是switch不可以判断区间

  2. const修饰指针有三种情况

    • const修饰指针 — 常量指针
    • const修饰常量 — 指针常量
    • const即修饰指针，又修饰常量

  3. 指针和函数

    • 利用指针作函数参数，可以修改实参的值

  4. 结构体指针

    • 利用操作符 -> 可以通过结构体指针访问结构体属性

  5. 结构体做函数参数

    • 传递方式有两种：值传递，地址传递
